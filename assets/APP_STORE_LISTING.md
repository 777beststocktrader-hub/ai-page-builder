### How It Works

1. **Describe your page** — Tell PageGenie what you want in one sentence
2. **AI builds it instantly** — Full landing page with hero, features, pricing, testimonials, FAQ, and more
3. **Customize with clicks** — Drag sections, edit text inline, pick colors and fonts
4. **Publish to your store** — One click publishes directly to your Shopify store

### Key Features

**AI Page Generation**
- Generate a complete, ready-to-publish landing page from a single sentence
- AI rewrites any section on demand — just describe what you want
- Polish entire page for consistency with one click
- Translate to 18 languages instantly

**28+ Professional Block Types**
- Hero, Features, Pricing, Testimonials, FAQ, Stats, CTA, Steps, Team, Gallery, Timeline, Countdown, Video, Newsletter, Contact Form, Custom HTML, and more

**8 Proven Page Templates**
- SaaS, Agency, Product, Restaurant, Portfolio, Mobile App, E-Commerce, Blog

**Design Tools**
- 9 theme presets + custom brand colors
- AI Brand Palette Generator — describe your brand, get 3 color + font options
- 10 Google Fonts, spacing control (compact/normal/spacious)
- Per-section background: solid color or 6 gradient presets
- Mobile / tablet / desktop preview

**Conversion & SEO**
- Live SEO Score — instant A–D grade as you build
- AI CRO Analysis — actionable tips to improve conversion
- A/B Headline Testing — 3 variants (outcome, pain-point, social proof angles)
- Schema.org JSON-LD + OG/Twitter meta tags in export

**Productivity**
- Drag-and-drop reorder sections
- Inline text editing — click any text to edit
- Undo / Redo (Ctrl+Z / Ctrl+Y)
- Block search (Ctrl+F), duplicate (Ctrl+D)
- Right-click context menu: lock, hide, duplicate, move
- AI block suggestions between sections
- Import page style from any URL
- Version history — save and restore up to 20 named snapshots

**Multi-Project Management**
- Create and manage multiple page projects
- Duplicate projects for fast iteration
- Publish as permanent hosted web pages with view tracking
- Share 48-hour preview links
